  kubectl port-forward svc/{{ .Release.Name }}-frontend 3000:{{ .Values.frontend.service.port }} -n {{ .Release.Namespace }}

Then open http://localhost:3000

To set API keys after install:

  kubectl create secret generic {{ .Release.Name }}-api-keys \
    --from-literal=OPENAI_API_KEY=sk-... \
    --from-literal=ANTHROPIC_API_KEY=sk-ant-... \
    -n {{ .Release.Namespace }} \
    --dry-run=client -o yaml | kubectl apply -f -

  kubectl rollout restart deployment/{{ .Release.Name }}-backend -n {{ .Release.Namespace }}
